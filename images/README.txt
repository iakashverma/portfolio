To use your profile image:

1. Save your profile photo as "profile.jpg" in this images folder
2. The image should ideally be square or portrait oriented
3. Recommended size: At least 800x800 pixels for best quality

The website is already configured to display: images/profile.jpg

Your image will appear in the About Me section with:
- Rounded corners (20px border radius)
- Shadow effect for depth
- Hover zoom effect (scales to 105% on hover)
- Maximum width of 400px
